Function filterOutType

  • filterOutType(tags, type): GeoTags[]

  • Filters out tags of a specific type from an array of tags.

    Parameters

    • tags: GeoTags[]

      The array of geotags to be filtered.

    • type: string

      The type of tag to be filtered out.

    Returns GeoTags[]

    A new array with the specified type of tags removed.

    This utility function filters out tags from an array based on the provided type. It iterates over the array and excludes any tags where the third element (the type identifier) matches the specified type. This is useful for removing specific types of geotags from a list of various tags.
